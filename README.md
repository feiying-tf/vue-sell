# vue-sell
仿外卖app

# 功能
1. 利用betterScroll实现各页面的滚动；
2. 利用vue-resourse实现ajax请求；
3. 通过本地存储实现收藏功能；
4. 通过模块坏，将多个组件进行复用；
5. 实现菜单与菜品的滑动互联；
6. 实现多种评论的信息筛选；
7. 实现加入购物车的小球动画，以及对应的相关计算；
8. 实现各页面的切换动画；

# 效果
## 商家介绍页面
1 点击头部商家介绍的公告和优惠的个数，即可进入到商家介绍页面；
